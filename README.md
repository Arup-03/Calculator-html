# 🧮 Simple Calculator Web App

A basic and responsive calculator built using **HTML, CSS, and JavaScript**.
This project performs simple arithmetic operations like addition, subtraction, multiplication, and division.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧹 Clear button (C)
* ⚠️ Error handling
* 🎨 Clean and simple UI

---

## 📂 Project Structure

```
calculator/
│── index.html   # Main structure
│── style.css    # Styling and layout
│── script.js    # Functionality (logic)
```

---

## ▶️ How to Run

1. Download or clone this repository
2. Open the project folder
3. Double-click on **index.html**
4. Run in your browser

---

## 💡 How It Works

* User clicks buttons to input numbers
* JavaScript stores input in display
* `eval()` function calculates the result
* Output is shown instantly on screen
